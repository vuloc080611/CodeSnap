## ✨ Why CodeSnap?

Every developer shares code snippets on Twitter, LinkedIn, blogs, or documentation. Existing tools are either closed‑source, watermarked, or require an internet connection.  
**CodeSnap** gives you:

- 🔒 **Self‑hosted** – no limits, no watermarks, your data stays local.
- 🎨 **Multiple themes** – VS Code Dark, Light, Night Owl, High Contrast, and more.
- 🖼️ **Custom backgrounds** – solid, gradients, or transparent PNGs for overlays.
- 📸 **High‑res export** – 2x pixel ratio, perfect for retina screens.
- ⌨️ **Monaco Editor** – the same editor that powers VS Code, with syntax highlighting.
- ⚡ **One‑click deploy** – free on Vercel.
